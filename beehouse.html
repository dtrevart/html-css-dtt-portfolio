Bee House (Game Scale)

I love bees, and there isn't a type of bee I love more then a Minecraft bee. When I saw the LEGO Minecraft The Bee Cottage set for the first time, I knew I had to get it. However, when I actually built it and had the model in hand, I was let down.

This was my first big LEGO project. I wanted to make something that was accurate the game and resembled the mob far more accurately. I had tackled making instructions before, but this was a bit different. I knew I was going to have to divide the model in half, and build each section in seperate files. Since I had built it all in one, it wasn't too bad copying stuff over, but making subassemblies, something I had never done before, was quite tricky at first. It came to me with time, however, and I think this still might be one of the easier to follow instructions I've made!
