Minecraft House Diorama

This is the first LEGO project that I really took full creative liberty on. A lot of my other projects have been based on something, whether it be recreating something or scaling something up, I've been using those to learn how to use the software. As someone who displays LEGO more than play with it, I wanted a little display I could put on my desk. It ended up a tad bit bigger than I was expecting, but it still fits nice on my desk! I looked up some Minecraft base inspiration, then did some building in game, using those as reference for my LEGO creation!

One of the things I'm most proud of about this build are the techniques. From the barrels having a backwards 2x2 jumper to add texture, the paintings sticking out ever so slightly along with the carpet, and getting the flower pot to line up to the center of the block with the limited parts in old brown, there are a lot of fun little techniques to help get the perfect look!

Something I decided I wanted to add after the fact was lighting. I thought if I were able to light up the lantern, it would take the build to the next level, and I was right! Thanks to Brick Loot, I was able to find a 1x1 transparent yellow plate that I could throw into my lantern, weave the wires around, and plug in! I'm super proud of how this turned out, and it has just made me more excited to create more in the future!
